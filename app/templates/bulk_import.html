CSV Format Specification

Column Required Format Example
name ✓ Yes String (max 255 chars) Web Server 1
ip_address ✓ Yes IPv4 or IPv6 192.168.1.1 or 8.8.8.8
group_name ✗ No String (max 100 chars) KSO_FH, KSO_HW, KSO_ZTE
tags ✗ No Semicolon-separated (;) web;production or dns;external
ping_interval ✗ No Integer (seconds) 60 (default), 30, 120

Notes

  • Max 5000 servers per import (split if needed)
  • Tags are case-sensitive
  • Use semicolons (;) to separate multiple tags: web;production;critical
  • Ping interval default: 60 seconds
  • All imported servers are active by default
  • Duplicate IPs will be created (manually check before importing)
